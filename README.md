# 🍽️ Tasty Bites

A Flutter-based food ordering application with Firebase integration.

## 📱 Features

- User Authentication
- Browse Food Menu
- Search Food Items
- Add to Cart
- Firebase Integration
- Responsive UI

## 🛠️ Built With

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage

## 📂 Project Structure

```
lib/
├── models/
├── screens/
├── widgets/
├── services/
└── main.dart
```

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Rashmi-12567/tastybytes.git
```

Navigate to the project

```bash
cd tastybytes
```

Install dependencies

```bash
flutter pub get
```

Run the project

```bash
flutter run
```
